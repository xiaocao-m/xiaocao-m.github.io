HTML基础

HTML指的是超文本标记语言，他是一种用来描述网页的一种语言。标记语言是一套标记标签

web标准的构成:

结构 结构用于对网页元素进行整理和分类(HTML)
表现 表现用于设置网页元素的版式、颜色、大小等外观样式(CSS)
行为 行为指的是网页模型的定义以及交互的编写(JAVAscript)

HTML 语法规范

  1. 所有标签都需要写在< >里面，大部分是成对出现。

  2. 单个的标签<br>表示换行

  3. 标签之间的关系

    • 包含关系

    • 并列关系

链接的id属性

id 属性可用于创建一个 HTML 文档书签。

提示: 书签不会以任何特殊方式显示，即在 HTML 页面中是不显示的，所以对于读者来说是隐藏的。

html图片

img是单标签

属性：

  • src : source图片路径，可以是相对、绝对、网络路径

  • alt ： 图片无法正常显示时的代替文本

  • width ： 规定图片的宽度

  • height ：规定图像的高度

  • title ：鼠标停留在图片上时提示的文本

type属性

  • 1 表示用数字123
  • a 表示用小写字母abc
  • A 表示用大写字母
  • i 表示小写罗马数字i,ii,iii
  • I 表示大写罗马数字I，II，III
